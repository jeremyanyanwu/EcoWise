# EcoWise
EcoWise is an intelligent system that accurately calculates an individual’s daily carbon footprint from lifestyle habits and provides personalized, actionable recommendations to reduce it
# EcoWise - AI Carbon Footprint Tracker

## How to run (takes < 2 minutes)
1. Open the Colab notebook: https://colab.research.google.com/drive/1zK8vJ5iX9kL3mPqR8sT2vN7uF4gH6jK9
2. Click "Runtime → Run all"
3. The dataset is downloaded automatically
4. Model trains in ~8 seconds
5. Results, SHAP plots, and predictions appear instantly

## Files
- `Carbon Emission.csv` → dataset
- `ecowise_lightgbm.ipynb` → complete notebook
- `carbon_model.txt` → trained LightGBM model
- `requirements.txt` → pip install -r requirements.txt
